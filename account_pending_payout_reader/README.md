# Usage

## Requirements

Python3
Running a polkadot node locally
Running a local api server through substrate-api-sidecar

`python3 payout_reader.py -a 14Y5c5hu5rC2MV8RhYWvJXpzm6fBECbGtU1VEr1GQxU92pTB -d 1`
